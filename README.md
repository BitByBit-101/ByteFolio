## 🌟 About Bytefolio

**Bytefolio** is a modern, responsive, and visually captivating **portfolio homepage template** designed for developers, creatives, and designers. Whether you're building a personal brand, job-hunting, or simply want a sleek digital presence — Bytefolio helps you start strong.

It’s fun, engaging, and full of personality — featuring **3D visuals, fluid animations**, and clean, semantic code.
Perfect for:
- Frontend developers
- UI/UX designers
- Creative technologists
- Anyone who wants their portfolio to feel **alive, interactive, and memorable** ✨

---

## 🧠 Features

- 🐡 **Interactive 3D Pufferfish** powered by Spline
- 🌀 Smooth **scroll-based animations** with GSAP
- 🎞 **Framer Motion** for elegant motion UI
- 🎯 **Fully responsive** for all screen sizes
- 🔁 Reusable animation components
- 🎨 **Custom cursor** and visual effects
- 🛠️ Easy to maintain and extend

---

## 🛠 Tech Stack

- **React** – Component-based UI
- **Framer Motion** – Animation library for React
- **GSAP (GreenSock)** – ScrollTrigger-based animation
- **Spline** – 3D object rendering and interaction
- **Tailwind CSS** – Utility-first styling
- **Vite** – Fast dev server and bundler
