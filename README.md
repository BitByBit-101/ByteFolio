## 🛠️ Tech Stack

- ⚛️ **React** — UI component library
- 🎞 **Framer Motion** — Declarative animations
- 🌊 **GSAP** — Scroll-triggered animations
- 🌀 **Spline** — Real-time 3D scenes
- 💨 **Tailwind CSS** — Utility-first styling
- ⚡ **Vite** — Lightning-fast dev server

---

## ✨ Features

- 🎮 Interactive 3D pufferfish using Spline
- 📱 Fully responsive layout (mobile to desktop)
- 🔁 Scroll-based and in-view animations
- 🧠 Smooth entrance effects with Framer Motion
- 🔗 Reusable animation wrappers and custom cursor
